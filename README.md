Cloud-based torrent downloader using Python and libtorrent on Google Colab, enabling direct downloads of torrent files and magnet links to Google Drive.

---

## 🚀 Features

* Direct download to mounted Google Drive
* Supports both **.torrent files** and **magnet links**
* Runs entirely on **Google Colab (no local setup needed)**
* Real-time progress tracking using interactive widgets
* Simple and easy-to-use interface

---

## 🛠️ Tech Stack

* Python
* libtorrent
* Google Colab
* Google Drive

---

## 📌 How It Works

1. Mount Google Drive in Google Colab
2. Upload a `.torrent` file or enter a magnet link
3. Files are downloaded directly to your Google Drive
4. Track download progress in real-time

---

## ▶️ Usage

1. Open the notebook in **Google Colab**
2. Run all cells step by step
3. Mount your Google Drive
4. Upload torrent file or paste magnet link
5. Monitor download progress

---

## 📂 Output

Downloaded files are saved in:

```
/content/drive/My Drive/Torrent
```
